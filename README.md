# Movie Explorer
 
This is my graded project for web development. The idea is to build a website where you can look up movies, search for them, and filter or sort them in different ways.
 
## What is this project?
 
Basically its a movie browsing app. You can search for a movie by name, filter by genre or rating, and sort by popularity or release date. The movie data comes from an API called TMDB (The Movie Database).
 
I picked this idea because movies are something everyone relates to and the TMDB API has a lot of data to work with which makes it easier to implement features like search and filtering.
 
## API being used
 
TMDB API - https://www.themoviedb.org/
 
The main endpoints I plan to use:
- /movie/popular (to load movies on the homepage)
- /search/movie (for the search feature)
- /genre/movie/list (to get the list of genres for filtering)
 
You need a free API key from their website to use it.
 
## Features I plan to build
 
- Search bar to find movies by name
- Filter movies by genre
- Filter by minimum rating
- Sort by popularity or release date
- Show each movie as a card with its poster, title, year and rating
- Maybe a detail page when you click on a movie (if I have time)
 
## How to run it
 
1. Clone or download the repo
2. Get a free API key from https://www.themoviedb.org/
3. Open app.js and paste your key where it says YOUR_API_KEY
4. Open index.html in a browser and it should work
 
No installations needed, its just plain HTML CSS and JS.
