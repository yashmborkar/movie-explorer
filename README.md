# Movie Explorer 🎬

A web application to search and explore movies using the OMDb API.

## 🚀 Features

* Search movies by title
* Sort movies (A-Z, Z-A)
* Favorite movies (interactive button)
* Dark/Light mode toggle
* Responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OMDb API

## ⚙️ How to Run

1. Clone the repository
2. Open `index.html` in browser
3. Enter a movie name and search
